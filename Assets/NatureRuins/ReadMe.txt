A stylized ancient ruins environment!
The Package includes 20 models.
 - There are 5 types of bricks to create your own constractions.
 - 1 Grass mesh.
 - 2 Vine Meshes.
 - 9 Brick Constractions.
 - 3 Colons[1 Big , 1 Small , 1 Base].
NOTE!!! This Package uses Unity's Build-in Shaders EXCEPT "Grass" and "Leaves" materials.For that porpose a Custom shader named "Nature" is included!!
If you want to transition to a URP or HDRP Project ,please make sure to create your own Shaders.
